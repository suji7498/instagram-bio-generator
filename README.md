# Instagram Bio Generator

A simple tool to generate cool and stylish fonts for your Instagram bio.

## Features

- Generate multiple font styles for your Instagram bio
- Copy fonts to clipboard with one click
- No installation required - works directly in your browser
- Mobile-friendly design

## How to Use

1. Enter your text in the input field
2. Click "Generate Fonts"
3. Browse through the different font styles
4. Click "Copy" button to copy a specific style to your clipboard
5. Click "Copy All to Clipboard" to copy all styles at once

## Deployment

This site is deployed using GitHub Pages. The simple version only requires an `index.html` file and this `README.md` file.
